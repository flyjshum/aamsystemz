Тестовое задание на позицию junior-разработчик

    Используемый стек технологий:
  • Backend: Java + Spring Boot. REST-контроллер для выполнения операций CRUD и поиска;
  • База данных: MySQL. ORM: Hibernate. Структура таблиц создается автоматичкески из сущности (POJO Product) при добавлении первого продукта. Предварительно на сервере MySQL необходимо создать БД test. Параметры для подключения к БД (Имя БД, Логин, пароль) в файле resources/application.properties;
  • Frontend: HTML5 - для шаблона страницы, JQuery - для отправки ассинхроного запроса к backend, обработки ответа в формате JSON, парсинга ответа и динамическое построение DOM страницы. Оформление с помощью CSS;
  • Обработка ошибок, логирование, интерфейс - минимальны. Только для демонстрации работы Backend.
